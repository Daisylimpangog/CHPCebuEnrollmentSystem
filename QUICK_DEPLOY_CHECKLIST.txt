================================================================================
COMMON ISSUES & SOLUTIONS:
================================================================================

Issue: "Cannot connect to database"
Solution: Check config.php has correct credentials from cPanel MySQL section

Issue: "No such file or directory" 
Solution: Verify all files uploaded to public_html folder

Issue: "Permission denied"
Solution: Set file permissions to 644, folder permissions to 755

Issue: "Session not working"
Solution: Contact hosting support - may need session path adjustment

Issue: "Forms not submitting"
Solution: Check error logs in cPanel, verify database user has INSERT privilege
